# Contributing
When you're contributing to this project, follow these rules:
* Do not use any proprietary services (Google Play Services, etc). Instead, use open-source alternatives, such as OpenStreetMap, for example.
* Use the Kotlin programming language. Java is not allowed
* Use Jetpack Compose and Material 3 for GUI elements
## Contributer License Agreement

By submitting a Pull Request, Issue, or any form of contribution to LibreAuto, you agree to the following:

1. **License Grant:** You grant the LibreAuto project indefinite, free use of your contribution with or without credit
2. **Ownership: You agree that you will still own your contribution, but allow the LibreAuto project to use it
3. **No Retraction:** You understand that once a contribution is merged, you may not revoke it
