# Contributing
When you're contributing to this project, follow these rules:
* Do not use any proprietary services (Google Play Services, etc). Instead, use open-source alternatives, such as OpenStreetMap.
* Use the Kotlin programming language. Java is not allowed
* Use Jetpack Compose and Material 3 for GUI elements
